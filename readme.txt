Ejecutando runEjeX.bat se ejecuta el ejemplo nro. X en el simulador.
El batch run.bat es un archivo que ejecuta la simulación de manera paramétrica y es utilizado por los runEjeX.

Si se quieren crear nuevos ejemplos, es conveniente utilizar el utilitario map2val. Leer el map2val readme para entender el funcionamiento de la aplicación. Hay que tener cuidado en crear laberintos conexos, ya que los automatas están programados para escapar de laberintos conexos.

Convención de colores.
En azul se encuentra la presa.
En tonos de grises el olor.
En verde, el predador.
En rojo el laberinto
